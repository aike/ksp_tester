How To Run
  Testcase scripts require TestSender.txt ,TestCommon.txt and TestHelper.txt.  
  Therefore copy them to same folder before recompile testcases.

  Compile the testcases by Nil's KScript Editor(http://nilsliberg.se/ksp/).
  Paste the compiled testcase to KONTAKT *multi script* window.

  Note: If you use precompiled .nkp testcase files, copy the files into multiscript folder
        before load them. ( Native Instruments/Kontakt 4/presets/multiscripts/ )
